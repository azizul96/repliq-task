# Project Description

  - e-Shop is an e-commerce platform, I have kept all the routes open so that you can browse without any interruption.
  - I tried to create all the views including Login and Registration, user can registers using a special Bangladeshi phone number and password during registration,  same information obtain a JWT and then using same phone number and password user can log in to the app.
  - I use mongoDB to store all the data.
